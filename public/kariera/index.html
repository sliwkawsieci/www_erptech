Stażysta Python - Frappe/ERPnext

Staż Zdalnie/Hybrydowo Python

Opublikowano: 7 listopada 2025

O stanowisku

Poszukujemy ambitnego stażysty, który chce rozwijać swoje umiejętności w zakresie programowania w Pythonie oraz poznać framework Frappe i system ERPnext. To doskonała okazja, aby zdobyć praktyczne doświadczenie w pracy z nowoczesnym systemem ERP open source.

Czego oczekujemy?

  • Podstawowa znajomość języka Python
  • Chęć nauki i rozwoju w obszarze systemów ERP
  • Znajomość podstaw HTML, CSS, JavaScript będzie dodatkowym atutem
  • Umiejętność pracy w zespole
  • Komunikatywność i zaangażowanie
  • Znajomość języka angielskiego w stopniu umożliwiającym czytanie dokumentacji technicznej

Co oferujemy?

  • Praktyczne doświadczenie w pracy z systemem ERPnext i frameworkiem Frappe
  • Mentoring ze strony doświadczonych programistów
  • Możliwość pracy zdalnej lub hybrydowej
  • Elastyczne godziny pracy
  • Przyjazną atmosferę pracy
  • Możliwość zatrudnienia po zakończeniu stażu
  • Udział w realnych projektach dla klientów

Zakres obowiązków

  • Wsparcie w rozwoju i customizacji systemu ERPnext
  • Tworzenie i modyfikacja DocTypes w Frappe
  • Pisanie skryptów automatyzujących procesy biznesowe
  • Tworzenie raportów i dashboardów
  • Testowanie i debugowanie aplikacji
  • Dokumentowanie kodu i rozwiązań

Technologie, z którymi będziesz pracować

Python Frappe Framework ERPnext JavaScript MariaDB Git Linux
